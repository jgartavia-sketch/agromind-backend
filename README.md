# AgroMind CR Backend

Backend y API de AgroMind CR, una plataforma enfocada en gestión agrícola, procesos productivos, automatización y administración inteligente de fincas.

---

## Funcionalidades

- API REST para frontend web
- Autenticación de usuarios
- Gestión de usuarios y sesiones
- Persistencia de datos agrícolas
- Gestión de fincas, zonas, tareas y procesos productivos
- Base preparada para asistencia con inteligencia artificial
- Base preparada para integración climática
- Base preparada para control e integración futura de dispositivos automáticos
- Arquitectura escalable para módulos agrícolas, financieros y operativos

---

## Tecnologías

- Node.js
- Express
- JavaScript
- PostgreSQL
- Prisma
- JWT
- Render
- Git
